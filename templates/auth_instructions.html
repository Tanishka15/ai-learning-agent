Google Classroom Authentication

Due to OAuth redirect restrictions, you need to follow these steps to authenticate with Google Classroom:

Step 1: Run Authentication Tool

Open a terminal in your project directory and run:

python classroom_auth_cli.py

This will open a browser window for you to authenticate with Google.

Step 2: Complete Google Authentication

In the browser window that opens:

  • Sign in with your Google account
  • Grant the requested permissions for Google Classroom access
  • Wait for the success message

Step 3: Return to the Application

Once authentication is complete:

Return to Application

Your authentication token has been saved and will be used automatically.
